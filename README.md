# ai_code_explainer
# AI Code Explainer for Beginners

## Problem
Many beginners struggle to understand programming code.

## Solution
This project is an AI-powered code explainer that helps users understand code in simple language.

## Features
- Explains code in simple terms
- Beginner-friendly interface
- Instant output

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript

## How to Run
1. Install Python
2. Run app.py
3. Open browser at http://127.0.0.1:5000/

## Future Scope
- Add real AI integration
- Support more languages
- Improve UI/UX
